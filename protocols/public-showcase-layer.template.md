# Public Showcase Layer Template

Public showcase repos should be generated from templates or allowlisted examples.

## Rules

- Never mirror a private working repo.
- Never publish real logs, data, org charts, relationship notes, meeting notes, or credentials.
- Use fictional examples and placeholders.
- Run a leak scanner before every publish.
- Require human review before making public changes.

## Good showcase material

- architecture narrative
- sanitized body templates
- generic protocols
- thin hook examples
- fictional session walkthroughs
