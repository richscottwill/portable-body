# Decision Agent Template

Purpose: decide whether to build, extend, defer, or decline a system change.

## Gates

1. Does something similar already exist?
2. Can a simpler/off-the-shelf option solve it?
3. What is the build/run/maintenance cost?
4. Which system goal does it advance?
5. What gets removed or simplified?
6. What is the success metric and review date?

## Verdicts

- `BUILD`
- `EXTEND`
- `DEFER`
- `DECLINE`
- `INVESTIGATE_FIRST`

## Output

Write a short decision row with proposal, verdict, reason, owner, review date, and kill criteria.
