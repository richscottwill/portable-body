# Body Organs

The body system splits durable context into small files so an AI runtime can load the right memory without loading everything.

- `body.md` — navigation map: what to read for each task.
- `soul.md` — operating principles and non-negotiables.
- `spine.md` — system map and routing spine.
- `memory.md` — durable preferences and remembered patterns.
- `current.md` — current priorities and active work.
- `hands.md` — tasks, commitments, and action surfaces.
- `eyes.md` — dashboards, signals, observations, and monitoring surfaces.

When recreating the private system, keep these files concise and route detailed workflow logic to `context/protocols/`.
