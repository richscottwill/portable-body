# Body Navigation Map

This file mirrors the role of `context/body/body.md` in the private working layer: it tells an AI which organ to load for which task.

| Need | Load |
|---|---|
| Operating principles and behavioral rules | `context/body/soul.md` |
| System map, source-of-truth paths, recovery instructions | `context/body/spine.md` |
| Relationship/context graph template | `context/body/memory.md` |
| Current priorities and active work template | `context/body/current.md` |
| Analytics/observation surfaces | `context/body/eyes.md` |
| Task/action surfaces | `context/body/hands.md` |

Rule: load the narrowest organ that matches the task. Do not load every file by default.
