# Docs

Docs explain the architecture and provide generated inventories.

- `START-HERE.md` — reading path for a new human or AI runtime.
- `system-map.md` — visual architecture map.
- `recreate-agent-bridge-system.md` — detailed replication guide for another machine.
- `replication-checklist.md` — validation checklist.
- `architecture/` — design model and portability reasoning.
- `catalog/` — generated inventories of hooks, protocols, agents, and workflows.
- `examples/` — fictional examples of runs, audits, and outputs.
