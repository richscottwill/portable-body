# Hands Template

Purpose: action surface and guardrails.

## Safe action pattern

1. Read current state.
2. Draft proposed action.
3. Classify risk.
4. Ask for approval if external, irreversible, or high-stakes.
5. Log outcome.

## Example actions

- Create a draft, not a sent message.
- Write a local artifact, not an external post.
- Produce a checklist, not a destructive command.
