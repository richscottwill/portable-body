# Agents

Agent files define reusable specialist roles.

A useful agent contract includes:

- purpose,
- sources it may read,
- outputs it must write,
- decisions it may make,
- decisions it must escalate,
- quality bar,
- failure mode.

The private system uses multiple agent teams: wiki teams, eval teams, compression/curation agents, sync agents, and workflow specialists. Public examples keep the role structure without private people, projects, or data.
